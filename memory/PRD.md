# Трифолд-буклет — Государственный Военный Госпиталь

## Дата: 2026-03-16

## Задача
Создание 100% готового к печати HTML/CSS трифолд-буклета (6 панелей, A4 Landscape) для отделения ТКМ госпиталя к China Fashion Week в Санья.

## Аудитория
VIP-персоны, топ-модели, инвесторы на China Fashion Week.

## Реализовано
- Standalone HTML-буклет `/brochure.html` с print-ready CSS (@page 303×216mm, bleeds 3mm)
- 6 панелей: Лицевая обложка, Задняя обложка, Внутренний клапан, Философия, Диагностика, УТП
- Все фото пользователя: дрон-фото госпиталя, стол ТКМ, коллектив, иглы, 3D-позвонки, QR-код
- Прайс-лист процедур (из .odt) интегрирован в панель 5
- React-лэндинг с превью обеих сторон и кнопкой экспорта в PDF
- Контент верифицирован по hospital1946.ru и new.hospital1946.ru/needle-knife/
- Шрифты: Cormorant Garamond + Montserrat (Google Fonts)
- Палитра: #FFFFFF / #1A1C1E / #D3D3D3
